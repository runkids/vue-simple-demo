## 說明

使用 Vue + Vue-router + Vuex + Element 實做簡單的問答題功能
此為練習，如有寫法不好的地方，歡迎提 Issues 或 PR 謝謝！

## 執行

``` bash
git clone https://github.com/runkids/vue-simple-demo.git

cd  vue-simple-demo

npm install

npm run dev

```
